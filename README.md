## Project Description

A Profile Management System web application developed using Java Web Technology.
Uses Java DB (Derby) database built inside NetBeans.

## Features
- Save Profile to database
- View all profiles
- Search profile by Student ID or Name
- Delete profile

## Technologies
- HTML, CSS, Java Servlet, JSP, JavaBean, JDBC, Java DB (Derby)

---

## MVC Structure
- Model: ProfileBean.java
- View: index.html, profile.jsp, viewProfiles.jsp
- Controller: ProfileServlet.java
